🏨 Hotel Booking Application
📌 Description

This is a full-stack Hotel Booking Application built using  MongoDB, Express.js, Node.js, and EJS Template.
It allows users to seamlessly search, explore, and book hotel accommodations with a user-friendly interface and secure backend system.

✨ Features
🔐 User Authentication (Login / Signup)
🔎 Search and Browse Hotels
🏨 Create & Manage Hotel Listings
⭐ Ratings & Reviews System
🗺️ Map Integration for Location View
